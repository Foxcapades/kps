inline fun toList(): List<UShort>
Content copied to clipboard

Returns a new list containing the contents of this deque in order from head to tail.

The length of the returned list will be equal to size.

Example

val deq = Deque(6)
deq += [4, 5, 6]
deq.pushFront(3)
deq.pushFront(2)
deq.pushFront(1)

deq.toList() // List[1, 2, 3, 4, 5, 6]
